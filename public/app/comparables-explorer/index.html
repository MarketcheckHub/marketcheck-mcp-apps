About This App

Flexible comparable search by VIN or make/model/year. Shows active listings (up to 50) and sold listings (up to 25) with price distribution, stats, and optional price prediction. Adjustable search radius for broad or narrow market analysis.

How to Use

1. In the Browser (Demo or Live Data)

Click "Launch App" above. Enter your MarketCheck API key in the settings gear for live data, or explore with demo data.

2. As an MCP App (AI Assistants)

Add the MarketCheck MCP server to Claude, VS Code, or any MCP-compatible host:

{
  "mcpServers": {
    "marketcheck-apps": {
      "url": "https://apps.marketcheck.com/mcp?api_key=YOUR_KEY"
    }
  }
}

Then ask the AI to use the comparables-explorer tool.

3. Embed in Your Portal

<iframe
  src="https://apps.marketcheck.com/apps/comparables-explorer/dist/index.html?api_key=YOUR_KEY&embed=true"
  width="100%" height="700"
  style="border:none;border-radius:8px;"
></iframe>

4. Get a Free API Key

Sign up at developers.marketcheck.com to get a free API key and start testing.
