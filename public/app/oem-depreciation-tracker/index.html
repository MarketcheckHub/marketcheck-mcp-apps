What This App Does

Brand-level residual value dashboard for OEM product, pricing, and remarketing teams. The app pulls used vs new average sale prices from MarketCheck's Sold Vehicle Summary API for your brand and a chosen competitor set, computes a residual % (used / new) per model, and projects each model's 36-month depreciation curve. It surfaces a KPI strip (your portfolio's weighted residual vs the segment average, best/worst performers, model count), interactive depreciation curves with hover tooltips, a sortable model ranking table with residual badges, body-type segment benchmarks with your brand's positions highlighted, a state-level geographic retention heatmap, and Value Alert cards flagging models that are depreciating unusually fast (≤60%) or holding unusually well (≥75%). Filters let you slice by body type and state for regional or segment-specific deep-dives.

Who Can Use This

OEM Product Planners
See which nameplates hold value and which are leaking residual fast — input for next-generation product decisions and feature investment priorities.
Pricing & Incentive Teams
Identify models that need pricing/incentive intervention before residuals erode further. Compare your portfolio's residual position vs the segment-weighted market.
Remarketing & Captive Finance
Set residual value forecasts for lease programs grounded in actual market behavior, not just guidebook estimates. Spot regional outliers to tune state-level lease offers.
Competitive Strategy
Benchmark your depreciation curves against direct competitors in the same body-type segment. Flag where competitors are pulling ahead on value retention.
OEM Field & Region Managers
Use the geographic heatmap to spot states with weak retention for your brand and target dealer training, marketing, or incentive support to shore up resale strength.

URL Parameters

Pass parameters directly in the URL to pre-fill the form and auto-generate the report:

ParameterDescription
api_keyYour MarketCheck API key (Enterprise tier required for Sold Vehicle Summary access)
myBrandYour OEM brand — e.g., Toyota, Ford, Honda. Auto-fills the brand selector. Also accepts make.
competitorsComma-separated competitor brands — e.g., 'Honda,Ford,Hyundai'. Pre-selects the chips.
bodyTypeFilter to a single body type — SUV, Sedan, Truck, Coupe, Hatchback, or Van. Also accepts body_type.
state2-letter state code to scope analysis to one geography (e.g., CA, TX). Default is national.
embedSet to 'true' for embedded iframe mode (hides demo banner)

Build It Yourself

1. Get a Free API Key

Sign up at developers.marketcheck.com to get a free API key.

2. As an MCP App (AI Assistants)

Add the MarketCheck MCP server to Claude Desktop, VS Code, or any MCP-compatible host:

{
  "mcpServers": {
    "marketcheck-apps": {
      "url": "https://apps.marketcheck.com/mcp?api_key=YOUR_KEY"
    }
  }
}

Then ask the AI: "Use the oem-depreciation-tracker tool to how fast are your models losing value vs the competition?"

3. Embed as an iFrame

<iframe
  src="https://apps.marketcheck.com/apps/oem-depreciation-tracker/dist/index.html?api_key=YOUR_KEY&embed=true"
  width="100%" height="700"
  style="border:none;border-radius:8px;"
></iframe>

4. Build from Scratch

Use the API flow diagram above as your implementation blueprint. Call the APIs in the sequence shown, handle parallel calls with Promise.all(), and render the UI components listed in "What to Render".
